Music Mixer

This is a team-based assignment where we created a music mixer using HTML, CSS, and JavaScript. The JavaScript adds interactive features such as drag-and-drop functionality and a play button. Users can drag different music tracks onto the mixer, and each sample of music plays when the play button is clicked. This project combines design and interactivity to create a fun and engaging way to mix music!
